FUNDAMENTO // O QUE É IDE

O que é uma IDE?

Integrated Development Environment, ambiente de desenvolvimento integrado. É o programa onde você escreve, testa e publica código em uma só janela. Windsurf, Cursor, Antigravity e VS Code são IDEs.

  • Editor de código com destaque de sintaxe, autocomplete e atalhos.
  • Terminal integrado para rodar comandos sem sair da IDE.
  • Controle de versão com Git e GitHub plugados nativamente.
  • Chat de IA que enxerga todo o projeto e edita arquivos com você.

CONCEITOS // VERSIONAMENTO

Commit, Push e Pull

CommitRegistro de uma alteração no projeto. Uma foto salva do estado atual dos arquivos.
PushEnvio dos commits da sua máquina para o GitHub.
PullCaminho contrário. Traz do GitHub para a sua máquina as atualizações mais recentes do projeto.
